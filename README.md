# Gravity Flyby (宇宙フライバイ)

重力を利用した2D物理パズルゲームです。
宇宙船をスリングショット操作で発射し、惑星の重力をうまく利用して、障害物を避けながらゴールを目指しましょう。

## 🎮 遊び方

1. **狙う**: 画面上の任意の場所をドラッグして、発射の方向と強さを決めます。点線で予測軌道が表示されます。
2. **放つ**: マウスボタンを離すと宇宙船が発射されます。
3. **祈る**: 発射後は操作できません。重力に身を任せてゴール（オレンジ色のエリア）への到達を見守りましょう。

## 🌟 特徴

- **リアルタイム重力演算**: 惑星からの距離に応じた引力が宇宙船の軌道を曲げます。
- **全3ステージ**:
  - **Level 1**: 基本的なスイングバイ操作。
  - **Level 2**: 2つの惑星の間を縫うS字軌道。
  - **Level 3**: 巨大な重力源とトラップが待ち受ける高難易度ステージ。

## 🛠️ 技術スタック

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 (Animations, Variables)

## 📦 インストールと実行

特別なインストールは不要です。`index.html` を最新のWebブラウザで開くだけでプレイできます。
